\# SYNTH.AI // Intelligence Synthesis Engine 🧪



An autonomous research agent that navigates the live web to synthesize complex technical topics into executive briefings.



\##  Overview

SYNTH.AI bypasses traditional search friction by using an agentic "Reflect \& Refine" loop. It doesn't just find links; it reads them, verifies the data, and drafts a cited report.



\##  Tech Stack

\- \*\*Inference:\*\* Groq Cloud (Llama 3.3 70B)

\- \*\*Search:\*\* Tavily AI (Real-time Web Index)

\- \*\*Interface:\*\* Streamlit (Custom Glassmorphism CSS)

\- \*\*Logic:\*\* Python 3.12
